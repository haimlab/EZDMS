EZDMS

Easy DMS This is a Graphical user interface designed with tools to help create and interpret mutant plasmid libraires.
First Variable sites count the number of amino acid located in a fastq file and all the frequency of each combination of multiple variable sites.
Second Preferences calculates the enrichment ratio of the amino acids in the libraries after they are placed under selective pressure.
Finaly Build primers assistants in generating the basic primers required to make these mutant plasmid libraries.
Please click on one of the tools below to get started.

Variable Sites


This tool processes a FASTQ library and generates an Excel spreadsheet with the counts of all combinations of variable sites detected. Sequences can also be further subdivided using the barcode option, which creates an additional tab in the spreadsheet that counts only the sequences containing the specified barcode.

Amino Acid Preference


This tool compares the change in amino acid preferences between a pre-mutation and a post-mutation library at a specific site. It normalizes the mutations against the wild-type sequences for accurate analysis. The output format is an excel file with the normalized amino acid prefrences.

Build Primer


The tool creates the 4 primers needed to create a plasmid library from a reference sequence in a text file (txt).
